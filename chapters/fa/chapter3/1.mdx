# مقدمه

در [فصل ۲](/course/chapter2) نحوه استفاده از توکنایزرها و مدلهای پیش تعلیم را جهت انجام پیش بینی های جدید بررسی کردیم. اما چگونه میتوانید یک مدل پیش تعلیم را خودتان بازتنظیم کنید؟

{#if fw === 'pt'}

* چگونه یک داده بزرگ را از هاب تهیه کنید
* چگونه از API سطح بالای "Trainer" برای بازتنظیم مدل استفاده کنید
* چگونه یک چرخه تعلیم دلخواه درست کنید
* چگونه از کتابخانه Accelerate 🤗 برای اجرای چرخه تعلیم دلخواه در هر تنظیمات غیر متمرکزی استفاده کنید 

{:else}

* چگونه یک داده بزرگ را از هاب تهیه کنید
* چگونه از کراس برای بازتنظیم مدل استفاده کنید
* چگونه از کراس برای استخراج پیش بینی ها استفاده کنید
* چگونه از یک متریک دلخواه استفاده کنید

{/if}
